Types of tags

There are two types of tags in HTML:

1. Paired tags 2. Unpaired tags

1. Paired tags

Paired tags are tags that have an opening tag and a closing tag. The closing tag is the same as the opening tag, except that it has a forward slash (/) in front of it.

2. Unpaired tags

Unpaired tags are tags that do not have a closing tag. They are also called self-closing tags. They are used to insert media such as images, videos, and audio into a web page.
